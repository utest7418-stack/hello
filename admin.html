Add / change image or video links and turn slots on or off. Changes save directly to your GitHub repo.

1. Connect to your GitHub repo

Token banane ka tarika: GitHub → Settings → Developer settings → Fine-grained tokens → Generate new token → Repository access: sirf ye repo select karo → Permissions: Contents = Read and write → Generate. Token sirf is browser session me rakha jaata hai, kahin save nahi hota — tab band karne pe delete ho jaata hai.
